AI Fashion Assistant: A Web Application for Outfit Visualization Based on Weather Data Using OpenAI and Diffusion Models

Poster presentation at 2025학년 한국의류학회 춘계학술대회

Abstract

This study explores an AI-powered web application for personalized fashion styling based on real-time weather conditions and cultural context. The system combines OpenAI’s language model to generate weather-appropriate outfit descriptions with a diffusion model to visualize these recommendations as realistic fashion images. Unlike existing styling applications that rely on static catalogs or user-uploaded images, the proposed approach dynamically generates outfits using live weather data, location-based cultural considerations, and text-to-image synthesis. The application is implemented as a web-based system using Vue.js and FastAPI, enabling cross-platform access without installation. The results demonstrate that integrating large language models with diffusion models can produce context-aware, visually coherent styling suggestions that resemble real-world outfits. While high-quality image generation introduces latency challenges, the study highlights the potential of AI-driven styling tools for personalized user experiences, fashion content creation, and global brand communication.

Technologies

FastAPI, VueJS, OpenAI Chat GPT API, Stable Diffusion AUTOMATIC1111 API, OpenWeatherMap API
