Research

Development of E-commerce website for individual entrepreneur

Bachelor degree thesis

Abstract

This project presents the design and development of a full-featured custom e-commerce web platform created for a local retail brand The system was built as a vanila PHP web application, focusing on performance, usability, scalability, and business practicality rather than template-based or CMS solutions. The platform includes a complete customer experience flow: product catalog with categories and filtering, product pages, cart system, and order checkout, as well as a fully functional administrative dashboard for content and order management. The architecture follows the MVC (Model–View–Controller) pattern and uses a structured backend design with object-oriented programming, relational database modeling, and modular routing. The result is a scalable, production-ready web system that demonstrates full-stack development skills, system architecture design, and real business integration — from concept and research to deployment and testing.

Technologies

HTML, CSS, JavaScript, PHP, MySQL
